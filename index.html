O que é JavaScript?

JavaScript é uma linguagem de programação usada para tornar sites interativos e dinâmicos.

  • Cria animações e efeitos visuais
  • Responde a ações do usuário (cliques, teclas, etc.)
  • Valida formulários antes de enviar
  • Permite atualizar conteúdo sem recarregar a página

🔧 Exemplo: abrir um menu ao clicar em um botão.

O que é o DOM?

DOM significa Document Object Model (Modelo de Objeto do Documento).

🧠 Ele é a representação da página HTML em forma de uma estrutura que o JavaScript pode acessar e modificar.

🏠 Analogia da casa:

  • A casa é o HTML
  • Cada cômodo representa uma <div>, <header>, <section>
  • Os móveis são os conteúdos: <p>, <button>, <img>
  • A planta da casa é o DOM: descreve os cômodos, o que eles contêm e onde estão

🧾 Exemplo HTML:

<section>
  <p id="mensagem">Olá, visitante!</p>
</section>

👀 Como o JavaScript vê isso no DOM:

document.getElementById("mensagem").innerText = "Bem-vindo!";

👉 O JavaScript localiza o parágrafo com ID mensagem e altera seu conteúdo.

O que é uma Função?

Função é um bloco de código reutilizável que executa uma tarefa específica.

🍓 Analogia: Fazendo um suco de morango

  • 📦 Os parâmetros são os ingredientes: morango, água e açúcar
  • 🔄 A função é o liquidificador — ela processa os ingredientes
  • 🥤 O resultado é o suco pronto, que a função retorna

🔧 Exemplo em JavaScript:

function fazerSuco(fruta, agua, acucar) {
  return "Suco de " + fruta + " com " + agua + "ml de água e " + acucar + "g de açúcar.";
}

let resultado = fazerSuco("morango", 200, 20);
console.log(resultado); // Suco de morango com 200ml de água e 20g de açúcar.

📌 A função recebe valores, executa algo e devolve um resultado (o suco 🧃).

Resumo Final

  • JavaScript: torna a página web viva e interativa
  • DOM: é como uma planta da casa — descreve toda a estrutura que o JavaScript pode manipular
  • Funções: são como liquidificadores — recebem ingredientes, processam e entregam o resultado

🎓 Agora é hora de praticar no código!

Métodos úteis em JavaScript

1️⃣ split() – Dividindo uma string

🧩 Divide uma string em partes menores (como cortar uma frase em palavras).

let frase = "Olá mundo bonito";
let palavras = frase.split(" ");
console.log(palavras); // ["Olá", "mundo", "bonito"]

🔍 A string foi transformada em um array de palavras.

2️⃣ join() – Juntando elementos

🔗 Junta os itens de um array em uma string, usando um separador.

let palavras = ["Olá", "mundo", "bonito"];
let frase = palavras.join(" ");
console.log(frase); // "Olá mundo bonito"

📌 É o contrário do split.

3️⃣ sort() – Ordenando elementos

📚 Organiza os elementos de um array em ordem alfabética ou numérica.

let nomes = ["Carlos", "Ana", "Beatriz"];
nomes.sort();
console.log(nomes); // ["Ana", "Beatriz", "Carlos"]

🗂️ O array foi reorganizado em ordem crescente.
